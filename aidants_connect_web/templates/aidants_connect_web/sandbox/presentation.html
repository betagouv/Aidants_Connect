Utilisation du site Bac à Sable

Le site bac à sable Aidants Connect est une copie du site principal, retrouvez toutes les informations ci-dessous pour l’utiliser.

Étape 1

Se rendre sur le site bac à sable en cliquant sur le bouton ci-dessous :

Lien vers le site Bac à Sable

Étape 2

Renseigner son adresse mail (celle utilisée pour vous connecter à votre compte) et le token suivant :

123456

Attention, ce token (ou code à 6 chiffres) est fixe, ne change jamais et est utilisable à l’infini. Sur le site réel, le token que vous utilisez n’est valable que 60 secondes.

Étape 3

Vous pouvez dèsormais tester la création de mandats avec des personnes fictives.

Lors de la connexion France Connect, vous pouvez utiliser le fournisseur d’identité « Démonstration » avec l’un des identifiants fictifs suivants :

Table des identifiants
Identifiant Mot de passe
test 123
avec_nom_dusage 123
sans_nom_dusage 123

Étape 4

Créer et valider le mandat en utilisant de nouveau le token :

123456

Attention, ce token (ou code à 6 chiffres) est fixe, ne change jamais et est utilisable à l’infini. Sur le site réel, le token que vous utilisez n’est valable que 60 secondes.

Quelques remarques complémentaires :

  • Ce site bac à sable est une copie conforme du réel site Aidants Connect. Toutes les futures modifications et fonctionnalités apportées sur le réel site Aidants Connect seront mises en place sur le site bac à sable.
  • Il permet uniquement de simuler la création d’un mandat avec un usager. Un trop grand nombre de contraintes techniques ne nous permet pas de simuler le « faire pour le compte de ».
  • Sur le site Bac à sable, l’espace Aidant ainsi que l’espace Responsable (administrateur d’organisation) sont accessibles. Il vous est donc possible de tester la création de mandat, mais aussi l’ajout d’aidant.
