Site bac à sable

Le site bac à sable Aidants Connect est une copie du site principal. Celui-ci vous permettra de vous familiarisier avec la création de mandat.

Procédure pour se connecter sur le site bac à sable

  1. Se rendre sur ce lien : Bac à sable Aidants Connect

  2. Renseigner son adresse mail (celle utilisée pour vous connecter à votre compte) et le token suivant : 123456

    Attention, ce token (ou code à 6 chiffres) est fixe, ne change jamais et est utilisable à l’infini. Sur le site réel, le token que vous utilisez n’est valable que 60 secondes.

  3. Vous pouvez dès aujourd’hui tester la création de mandats avec des personnes fictives.

    Lors de la connexion France Connect, vous pouvez utiliser le fournisseur d’identité « Démonstration » avec l’un des identifiants fictifs suivants :

    Identifiant Mot de passe
    test 123
    sans_nom_dusage 123
    avec_nom_dusage 123

  4. Créer et valider le mandat en utilisant de nouveau le code : 123456

Quelques remarques complémentaires :

  • Ce site bac à sable est une copie conforme du réel site Aidants Connect. Toutes les futures modifications et fonctionnalités apportées sur le réel site Aidants Connect seront mises en place sur le site bac à sable.
  • Il permet uniquement de simuler la création d’un mandat avec un usager. Un trop grand nombre de contraintes techniques ne nous permet pas de simuler le « faire pour le compte de ».
  • Sur le site Bac à sable, l’espace Aidant ainsi que l’espace référent (administrateur d’organisation) sont accessibles. Il vous est donc possible de tester la création de mandat, mais aussi l’ajout d’aidant.
