Cette autorisation est conforme aux articles 1984 et suivants du Code civil. Les démarches seront accomplies sur internet, sur des sites utilisant FranceConnect ou sur d’autres sites. Elles peuvent être aussi accomplie par tous les moyens permis par l’administration (téléphone, papier, email, …).

Pour que {{ aidant.organisation }} puisse agir à ma place :

  • Je reconnais que l’aidant habilité par {{ aidant.organisation }} m’a rappelé l’objet de son intervention, et m’a informé sur la nécessité et l’utilité des informations collectées ;
  • J’autorise les aidants habilités par {{ aidant.organisation }} à utiliser mes données personnelles dans le cadre de ce mandat. Je sais que j’ai des droits sur les informations me concernant : accès, rectification, suppression.

Les aidants habilités par {{ aidant.organisation }} doivent :

  • effectuer les démarches listées dans ce document à ma place, à partir des informations que je leur ai données ;
  • collecter et conserver seulement les informations nécessaires aux démarches listées dans ce document ou à celles qui s’y rattachent ;
  • utiliser et communiquer seulement les informations nécessaires aux démarches listées dans ce document ou à celles qui s’y rattachent ;
  • m’informer et demander mon autorisation avant d’effectuer d’autres démarches que celle listées dans ce document ;
  • mettre à jour et supprimer l’ensemble de mes informations personnelles lorsqu’elles ne sont plus utiles ;
  • s’interdire de rendre publiques mes informations personnelles ;
  • prendre toutes les précautions pour assurer la sécurité de mes informations personnelles.

A partir du moment où un aidant habilité par {{ aidant.organisation }} réalise à ma place une des démarches listées dans ce document, il accepte de le faire dans les conditions décrites dans ce document.

Mon autorisation est donnée et acceptée {{ duree }}. Elle se termine avant si :

  • les démarches décrites ci-dessus sont réalisées ;
  • je décide de l’annuler ;
  • l’aidant habilité par {{ aidant.organisation }} décide de l’annuler ;
  • je donne mon autorisation à une autre personne qu’un aidant habilité par {{ aidant.organisation }}. Dans ce cas, je dois en informer le plus vite possible {{ aidant.organisation }}.

L'aidant habilité par {{ aidant.organisation }} doit accomplir à ma place les démarches listées dans ce document tant que le mandat est valable.
L’aidant habilité par {{ aidant.organisation }} pourra être responsable s’il ne respecte pas les conditions décrites dans ce document (article 1991 du Code civil.).
